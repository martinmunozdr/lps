Generalidades


Este programa manipula varios tipos de datos: numeros con decimales, enteros, textos,
valor de verdad (verdadero y falso), tambien pueden tener el valor VACIO
Tiene tambien estructuras de datos como son: listas, arreglos, diccionarios.
Soporta programacion orientada a objetos utilizando clases.
Tiene subrutinas, condicionales, bucles, sentencias para lectura y escritura de datos de pantalla.

Tiene un conjunto de operadores que le permite resolver expresiones.

Los identificadores(nombres de variables, arreglos y subrutinas) deben:
- Empezar con una letra
- Estar formados solo por numeros, guion bajo y letras
- No tener el mismo nombre de una palabra reservada

Tener en cuenta que este lenguaje distingue mayusculas de minusculas.
Los comentarios empiezan con el caracter # y son de una sola linea
Este lenguaje se puede usar para programacion web utilizando cgi, tambien puede conectarse con bases de
datos Sqlite, tiene subrutinas predefinidas para esto.


TIPOS DE DATO DEL LENGUAJE LPS:
ENTERO ejm: 24

NUMERO CON DECIMALES ejm: 34.78

VALOR DE VERDAD ejm: VERDADERO

TEXTO ejm: "prueba"

LISTA ejm: [1,"qqq",34.18,FALSO]

ARREGLO ejm: ARREGLO(8)

DICCIONARIO:
ejm:
d=DICCIONARIO()
d["qqq"]=34
d["ww"]="prueba"
