Variables


Las variables son dinamicas, es decir que pueden cambiar de valor e incluso de tipo en el transcurso del programa.

Tipos de datos posibles:
x=5 #numero entero
x=5.7 #numero con decimales
x=VERDADERO #valor de verdad
x=5<4 #FALSO, valor de verdad

x=LISTA() #creo una lista vacia

x =[4,3,"probando"] #creo una lista de 3 elementos
x = [x,"otro elemento"] #agrega un elemento a la lista x, si
#se hubiera ejecutado la anterior linea antes,
#x tendria ahora: 4,3,"probando","otro elemento"

x= VACIO #asigno el valor VACIO a x

arr=ARREGLO(3)
arr[1]=3
arr[2]=15.2
arr[3]="prueba"

x=DICCIONARIO() #creo un diccionario vacio
x["clave1"] =5 #creo clave y asigno valor
x["otraclave"] ="prueba" #creo clave y asigno valor
x["clave1"] = x["clave1"]*3 #triplico el valor de la clave1 del diccionario x

l = [13,2,30,"prueba"]
l[2,4] #sublista, dara [2,30,"prueba"]
t="hola mundo"
t[3,7] #dara: "la mu"
#si quisieramos cambiar el valor de t a una subcadena, podriamos usar
t=t[3,7]
